Map Keys

  • Set a horizontal and vertical map size, set a map name and file name.
  • Select a map tile type by left-clicking the tile type from the Map Keys section.
  • Left-Click or Left-Click and Drag across grids in the map to change those grids to the selected map tile type.
  • Right-Click a field to fill the contiguous fields of that clicked on map tile type to the selected type.
  • Click Save to create an output map file.
  • Click and Drag into the provided area to load an existing map.
